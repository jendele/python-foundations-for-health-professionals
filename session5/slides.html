§1 · Files

Open the door. Close it on your way out.

A file is bytes. To read text we need encoding. To read a table we need separator.

with guarantees the file closes — even if something crashes inside.

from pathlib import Path
DATA = Path("../data")

with open(DATA / "drg_catalog.csv",
          encoding="utf-8") as f:
    rows = f.readlines()

print(len(rows), "lines")
The lo-fi way — once, then pandas

§3 · NumPy

Arrays as typed memory.

A Python list holds anything — flexible but slow.

A NumPy array holds one type in contiguous memory — small, fast, the calculator's friend.

Mental model: list is a notebook with words and drawings. Array is graph paper with numbers in the squares.

import numpy as np
los = np.array([5, 3, 9, 4, 7, 2, 8])

los * 220             # vectorised: no for-loop
los > 5               # boolean mask
los[los > 5]          # filter by mask
los.mean(), los.std() # C under the hood
Vectorisation = speak in arrays, not in loops
§4 · Broadcasting

When shapes meet.

NumPy operates element-wise. If shapes match — straight arithmetic.

If they differ, NumPy broadcasts the smaller one across the larger.

"For all 320 encounters, multiply by efficiency factor 0.95" — one line, one number, one array.

cost  = np.array([3450,1820,8950,2380,7800])
reimb = np.array([3450,1820,8950,2380,7800])

margin = reimb - cost          # element-wise
target = reimb * 0.95          # scalar broadcasts
losing = (margin < 0).sum()    # aggregate
One line ↔ one statement of intent

§5 · Pandas Series

A NumPy array that remembers labels.

Series = 1D NumPy array + an index.

Every column of a DataFrame IS a Series.

Access by label, vectorise underneath. Best of both worlds.

ages = pd.Series(
    [67,54,82,39,71],
    index=["P0001","P0002","P0003","P0004","P0005"])

ages["P0003"]      # 82
ages > 65          # Series of bool
ages[ages > 65]    # filtered Series
(ages * 0.95).round(1)
Series = labelled array

§7 · Clinic

320 encounters → margin per case.

enc = pd.read_csv(DATA / "encounters.csv",
                  parse_dates=["admission_date","discharge_date"])
drg = pd.read_csv(DATA / "drg_catalog.csv")

# Join reimbursement onto each encounter:
enc = enc.merge(drg, on="drg_code", how="left")

# Vectorised margin — one line replaces yesterday's loop:
enc["margin_eur"] = enc["base_reimbursement_eur"] - enc["actual_cost_eur"]

losing = enc[enc["margin_eur"] < 0]
print(f"Loss-making: {len(losing)} of {len(enc)}")
print(f"Total negative margin: € {losing['margin_eur'].sum():,.2f}")
