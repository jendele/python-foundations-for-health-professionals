By the end of this session you will:

- write multi-stage conditional branches with `if` / `elif` / `else`
- iterate over a list with `for`, and over a condition with `while`
- use `break` and `continue` deliberately
- read (and write a simple) list comprehension
- classify a list of patient records against a clinical guideline
